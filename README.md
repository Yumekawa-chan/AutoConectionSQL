# AutoConectionSQL
東京電機大学のAJ科目"データベース言語SQL"のPostgresSQLサーバーに自動接続するためのpythonコードです。

# 使用条件
・授業で指定された方法でPstgresの設定がしてある

・Windows

・Pythonが導入されている(pip:pyautogui)

# 実行方法
terminalで該当フォルダーに行って

`python auto_sql.py`

この作業がめんどくさかったらショートカットでも作ってください。
